## Post (English version)

---

Most people use Claude Code to write code.

I used it to build an intelligence system with 11 specialized agents that follow the CIA Intelligence Cycle methodology.

Without writing a single line of code.

**What it does:**

Two modes:

1. **Corporate Intelligence** - Analyzes companies like an intelligence analyst: SWOT, Porter's Five Forces, Analysis of Competing Hypotheses, Red Team, and Scenario Planning. Output: executive report in DOCX.

2. **Person Intelligence** - Professional assessment of individuals with a mandatory ethical consent gate. Leadership Score, Influence Score, Professional SWOT, and 3 career trajectory scenarios. If the analysis lacks professional justification, the system stops before collecting a single data point.

**How it works:**

Configuration files (markdown + JSON) inside a `.claude/` folder:
- 11 subagents with specific roles, tools, and models
- 5 slash commands that orchestrate multi-agent workflows
- Hooks that automate pre/post analysis tasks
- Skills that activate when they detect patterns

Claude Code loads them automatically. Clone the repo, open `claude`, type `/cia-analyze Stripe`.

**Why it matters:**

Claude Code isn't just a development tool. It's a general-purpose agent platform.

With the same architecture you can build:
- Sales intelligence for prospect analysis
- Investment research for due diligence
- Talent assessment for executive roles
- Operational or regulatory risk assessment

No APIs, no dependencies, no infrastructure. Just declarative configuration.

Repo: https://github.com/hiveagents-dev/agentic-cia-intelligence-system
