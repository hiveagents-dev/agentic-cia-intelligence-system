# Scenario Planning Prompt (CIA Style)

Develop three strategic scenarios for {{COMPANY}} using intelligence community methodology.

## Instructions

Create Best Case, Most Likely, and Worst Case scenarios. Each must include:
- Probability assessment (must sum to ~100%)
- Specific conditions required
- Quantified impact where possible
- Early warning indicators to monitor

## Scenario Framework

### Best Case Scenario (Probability: X%)
| Element | Detail |
|---------|--------|
| **Conditions** | What must happen for this scenario to occur |
| **Timeline** | Expected timeframe |
| **Revenue Impact** | +X% growth / $X additional revenue |
| **Market Position** | How position changes |
| **Key Drivers** | 2-3 critical factors |
| **Early Indicators** | Signals this scenario is materializing |

### Most Likely Scenario (Probability: X%)
| Element | Detail |
|---------|--------|
| **Conditions** | Current trajectory continuation factors |
| **Timeline** | Expected timeframe |
| **Revenue Impact** | +/-X% change |
| **Market Position** | Expected evolution |
| **Key Drivers** | 2-3 critical factors |
| **Early Indicators** | Signals confirming this path |

### Worst Case Scenario (Probability: X%)
| Element | Detail |
|---------|--------|
| **Conditions** | What must go wrong |
| **Timeline** | Expected timeframe |
| **Revenue Impact** | -X% decline / $X loss |
| **Market Position** | How position deteriorates |
| **Key Drivers** | 2-3 critical risk factors |
| **Early Indicators** | Warning signals to monitor |
